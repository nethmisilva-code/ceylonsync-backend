# CeylonSync Backend

Backend API for Tea Factory Management System built with Node.js, Express, and MongoDB.

## Setup
1. npm install
2. create .env
3. npm run dev

## Scripts
- npm run dev
- npm start
- npm run seed:admin

## Main Modules
- Auth & Users
- Products
- Suppliers
- Raw Materials & Inventory
- Cart & Orders
- Payments & Invoices
- Employees & Payroll
- Reports
